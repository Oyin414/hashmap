# About Hashmaps

This Hashmap project is a good way for me to practice using and implementing my own Hashmaps. A Hashmap is a similar data structure to an
array but it uses keys and values like an object , the keys in the hashmap are internally converted to numerical indices whereas in an array the numerical indices usually depend on the order the array items were added in.

## How It's Made:

**Tech used:** JavaScript,webpack,jest

This project uses jest tests to make sure the methods written in my Hashmap class were working as designed.

## Lessons Learned:

During this project my main goal was to learn how to write and use Hashmaps , a Hashmap is very useful as it allows the lookup speed of the hashmap is very fast compared to other data structures so regardless of if you have 500 items or 500000 items , the Hashmap allows you to find the item you are looking for quickly.It was hard learning how to structure the linked lists inside the buckets and getting the set method to work properly and keep track of all the buckets and items in the bucket so it can decide when to double capacity.
